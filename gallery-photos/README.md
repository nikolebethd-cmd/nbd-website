# Gallery Photos

Drop your photos into the subfolder that matches the gallery `id` in `galleries.json`.

Example:
  gallery-photos/
  └── backcountry-2025/
      ├── DSC_0001.jpg
      ├── DSC_0002.jpg
      └── ...

Supported formats: JPG, JPEG, PNG, WEBP

The server reads this folder automatically — no config needed when you add photos.
To add a new gallery, add an entry to `galleries.json` and create the matching folder here.
